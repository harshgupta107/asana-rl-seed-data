# Asana Simulation Engine

## Overview
A Python-based simulation engine that generates high-fidelity seed data for an Asana workspace. This project serves as a foundational dataset for Reinforcement Learning (RL) environments.

## Directory Structure
- `src/generators`: Logic for creating Users, Projects, and Tasks.
- `src/models`: Database connection management.
- `schema.sql`: SQLite database definition.
